# Learning Path

My journey into low-level programming followed a structured path:

1. **C++ Fundamentals** - TheCherno's C++ tutorial series
2. **OpenGL Basics** - Understanding graphics pipeline, shaders, and rendering
3. **Hazel Engine** - Advanced game engine architecture (ongoing)

I also set up **Asahi Linux** on my MacBook to have a proper development environment for OpenGL work, learning Linux system administration along the way.
