# Mercedes-Benz G Class

A departure from hypercars, the G Class was a challenge in modeling boxy, angular shapes while maintaining visual interest.

The iconic Mercedes SUV required a different approach - focusing on sharp edges and flat surfaces rather than the flowing curves of supercars.
