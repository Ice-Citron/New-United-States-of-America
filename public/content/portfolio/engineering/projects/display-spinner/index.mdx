# Introduction

This is my iGCSE Design & Technology coursework project - a Persistence of Vision (POV) Display Spinner. The project combines 3D CAD modeling, laser cutting, Arduino programming, and electronics to create an interactive display system.

The display spinner uses the persistence of vision effect - when LEDs spin fast enough, they create the illusion of a solid image floating in mid-air. The system includes motion sensors for automatic activation, a real-time clock for scheduling, and motor speed control via potentiometer.

# Electronics & Programming

## Arduino System

The display spinner is controlled by an Arduino Mega with the following features:

**Sensors & Inputs:**
- 4x PIR motion sensors for automatic activation
- Potentiometer for motor speed control
- 4x Control buttons for mode selection
- Real-time clock (DS1302) for scheduled operation

**Outputs:**
- BTS7960 motor driver for PWM speed control
- 4x Relay modules for LED control
- LED array for POV display

**Smart Features:**
- Automatic activation when motion is detected
- Time-based pause duration (longer during day hours 8AM-8PM)
- Calibration sequence on startup
- Debounced button inputs

```cpp
// Key components used:
#include <ezButton.h>
#include <virtuabotixRTC.h>

// Motor control pins
int R_EN = 2;
int R_PWM = 3;
int L_EN = 4;

// PIR sensors on pins 5-8
// Relays on pins 9-12
// RTC on pins 50-52
```

# Manufacturing

## Laser Cutting

The main structure was manufactured using laser cutting on acrylic sheets. The SVG files were exported from Blender and used to precisely cut all the components.

## Assembly

The assembly process involved:
1. Laser cutting acrylic panels
2. Mounting the motor and bearings
3. Wiring the LED array with slip rings for power transfer
4. Installing the Arduino and sensor array
5. Programming and calibration
